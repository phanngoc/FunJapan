# {FJ-theme}

## Install
### Requirement
- Install Node.js and npm https://nodejs.org.
- Install bower and gulp globally:
```
npm install -g bower
```
```
npm install -g gulp
```
### Setup Develop Environment
Step1:
```
npm install
bower install
```

Step 2:
```
gulp dev
```


### Setup build
Step 1:
```
npm install
bower install
```

Step 2:
```
gulp build
```
